# 🤖 AI Toolbox – Text Processing Tool

> A Python-based text processing tool built with Google Gemini API, featuring summarization, email polishing, translation, code commenting, and todo classification.

## 🚀 Features
1. Summarizer: Input long text, output one-sentence summary
2. Email Polisher: Convert draft emails into formal, professional versions
3. Translator: Supports English, Chinese, Malay translation
4. Code Commentator: Automatically add detailed comments to code
5. Todo Classifier: Organize messy todo items by categories
